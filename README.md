## Python Study:
### Visual Studio Code常用快捷鍵:

## 功能	快捷鍵
1. 整行* 註解	Ctrl + K , Ctrl + C
2. 整行* 取消註解	Ctrl + K , Ctrl + U
3. 整行* 刪除	Ctrl + Shift + K
4. 整行* 上移 / 下移 一行	Alt + ↑ / ↓
5. 整行* 在下方 / 在上方 複製貼上	Alt + Shift + ↑ / ↓
6. Alt + ↑ 或 Alt + ↓ 快速移動整行程式碼
7. 縮排* Shift + Tab
8. 一次性更改變數名稱：游標移至變數 + F2
